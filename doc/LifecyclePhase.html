    class LifecyclePhase

    This model represents the pre-defined lifecycle phases. Lifecycles have many lifecycle phases; each project phase belongs to a lifecycle phase. Deliverable types are pre-defined for each lifecycle phase.

    Custom validator to check whether sequence number is within acceptable range The sequence numbering starts at 1. However, we need an extra placeholder for swapping sequence numbers. So, we use 0 as the temporary swap number.

    # File app/models/lifecycle_phase.rb, line 31
def sequence_number_range
  unless self.lifecycle.nil? || self.sequence_number.nil?
      if((self.sequence_number > lifecycle.lifecycle_phases.count+1) || (self.sequence_number < 0) )
         errors.add(:sequence_number, "must not exceed the number of lifecycle phases and must be greater than 0")
      end
  end
end

def unique_sequence_number
  unless self.lifecycle.nil?
      if(lifecycle.lifecycle_phases.find_all_by_sequence_number(self.sequence_number).count > 0)
         errors.add(:sequence_number, "must be unique for this lifecycle")
      end
  end
end

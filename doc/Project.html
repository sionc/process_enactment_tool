    class Project

    This model represents user-created projects. Each project is associated with a lifecycle, and has a collection of project phases.

    Returns the estimated effort for the Project by taking the sum of the estimated effort for ProjectPhase

def estimated_effort
  total_estimated_effort = 0
  project_phases.each do |phase|
    total_estimated_effort += phase.estimated_effort unless phase.nil?
  end

  total_estimated_effort
end
